Initialise Flexbox

  • Flexbox is initialised by adding the following command to a class or id.
  • In 'Flexbox Zombies', this command is placed inside of the crossbow {} class to arm the player's crossbow.

CSS to Initialise Flexbox:

display: flexbox
OR
display: inline-flex;

Notes:

  • inline-flex makes the flex container display inline.



Flexbox Direction

  • Flexbox direction is set horizontally, vertically or mirrored with the following CSS code.
  • In 'Flexbox Zombies', this command sets the direction of aim for the player's crossbow depending on facing direction and zombie approach direction.

CSS to Set Flexbox Direction:

flex-direction: row
flex-direction: row-reverse
flex-direction: column
flex-direction: column-reverse

Notes:

  • Default value: row (left->right)
  • Column sets direction from top to bottom
  • row-reverse sets right->left and column-reverse bottom->top



Justify Content

  • Justify content is used to define the alignment along the main axis of the flexbox using the following CSS code.
  • In 'Flexbox Zombies', this command sets the target finder of the crossbow to different positions on the field relative to flex-direction e.g. using justify-content:flex-end to target a zombie at the end of the row or column.

CSS to Justify Content (main-axis):

justify-content: flex-start
justify-content: flex-end
justify-content: center
justify-content: space-between
justify-content: space-around

Notes:

  • Default value: justify-content: flex-start
  • space-around will have gaps at the margins whereas space-between will have none



Align Items

  • Align Items is used to define the alignment along the cross axis of the flexbox (perpendicular to main-axis) using the following CSS.
  • In 'Flexbox Zombies', this command sets the target finder of the crossbow to different positions on the field relative to flex-direction e.g. using align-items: flex-start on a flex-direction: row will aim for zombies at the top of the field.

CSS to Align Items (cross-axis):

align-items: flex-start;
align-items: flex-end;
align-items: flex-stretch;
align-items: flex-center;

Notes:

  • Default value: stretch (fill container)



Align Self

  • Align Self is used to define the position of an anomalous value in a series of flex items.
  • In 'Flexbox Zombies', a new class is created in addition to the crossbow class for the zombie that is out of the axis of aiming or justify/stretch properties.
  • The player is then able to tartget zombies along the main alignment as well as the anomaly.

CSS to Align Self:

align-self: flex-start;
align-self: flex-end;
align-self: flex-stretch;
align-self: flex-center;

In 'Flexbox Zombies' where two are along the center of the field X-Axis and one is at the top:

crossbow {
display: flex;
justify-content: flex-end;
align-items: center;
}

.target:nth-of-type(2) {
align-self: flex-start;
}

Notes:

  • Default: flex-start



Flex Grow

  • The flex grow setting is about ratios, for example if we set the property to flex-grow: 1; items in the class will grow to fill the available space at the same rate.
  • Different values make items grow at different speed rations e.g.in two classes with flex-grow: 1; and flex-grow:2; two will take 2 pixels for 1's 1.
  • In 'Flexbox Zombies' when there is extra space, the target item will grow to fill it along the direction you are aiming at the ratios you declare.

CSS to flex-grow flex items:


flex-grow: n;

Using flex-grow in 'Flexbox Zombies' to aim at targets growing at a ratio of 3:1:


.target:nth-of-type(1) {
flex-grow: 3;
}

.target:nth-of-type(2) {
flex-grow: 1;
}

Notes:

  • Default value: flex-grow:0
  • Flex-grow is always assigned to targets, not crossbow.
  • Justify content setting doesn't do anything if there is growing happening because there is no available space.
  • It can be good practice to set all flex-grow properties to: 1 then override and adjust ratios for items growing faster than peers.
  • Sometimes you will use align-self: x; in conjunction with flex-grow e.g. stretch.



Flex Shrink

  • Where flex-grow deals with how items expand to fill available space, flex-shrink describes how items react when there's not enough space for them to fit.
  • This is defined using ratios similar to flex-grow e.g. an item with flex-shrink ratio of 3:1 will shrink 3 times faster than the default.
  • In 'Flexbox Zombies' - goos can both shrink and grow. If they don't shrink their flex-shrink is set to 0

CSS to flex-shrink flex items:

flex-shrink: 1;

Notes:

  • Default value: flex-shrink: 1;



Flex Basis

  • The flex-basis property specifies the initial length of a flexible item.
  • Note: If the element is not a flexible item, the flex-basis property has no effect.

CSS to initialise flexbox:

flex-basis: number;
flex-basis: auto;
flex-basis: initial;
flex-basis: inherit;

e.g.
div:nth-of-type(2) {
flex-basis: 100px;
}

Notes:



Order

  • By default, flex items are laid out in the source order.
  • However, the order property controls the order in which they appear in the flex container.

CSS to set order of flex items:

order: number
order: initial
order: inherit

e.g.
div#myRedDIV {order: 2;}
div#myBlueDIV {order: 4;}
div#myGreenDIV {order: 3;}
div#myPinkDIV {order: 1;}

Notes:



Flex Wrap

  • By default, flex items will all try to fit onto one line.
  • You can change that and allow the items to wrap as needed with this property.

CSS to flex wrap:

flex: nowrap
flex: wrap
flex: wrap-reverse

e.g.
div {
display: flex;
flex-wrap: wrap;
}

Notes:

  • Default value: nowrap (one line)



Align Content

  • This aligns a flex container’s lines within when there is extra space in the cross-axis.
  • It does this is a similar way as to how justify-content aligns indvidual items within main axis.

CSS to justify content along cross-axis:

align-content: flex-start
align-content: flex-end
align-content: center
align-content: space-between
align-content: space-around
align-content: space-evenly
align-content: stretch

Notes:

  • Default value:align-content: flex-start



Flex Shorthand

  • The three properties assigned to the .item class can be shortened in one line using the flex shorthand property.
  • You have to enter the values in this order:
    1. flex-grow
    2. flex-shrink
    3. flex-basis

CSS to initialise flexbox:

e.g.
.item { flex: 1 1 150px; }

Notes:

  • You can test code by resizing browser.
